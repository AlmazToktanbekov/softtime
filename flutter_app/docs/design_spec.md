## 1. ФИРМЕННЫЙ СТИЛЬ (Brand Identity)

- Слово **"Soft"** — чёрный, жирный, скруглённый шрифт

## 3. АНИМАЦИИ И ПЕРЕХОДЫ

### 3.1 Навигация (Flutter)

```dart
// Переход между экранами — плавный slide
PageTransition:
  type: SlideRight (вперёд) / SlideLeft (назад)
  duration: 280ms
  curve: Curves.easeInOutCubic

// Bottom sheet
BottomSheet:
  duration: 320ms
  curve: Curves.easeOutCubic
  
// Hero transitions на карточках
Hero(tag: 'employee_${user.id}')
```

### 3.2 Micro-анимации

```
Кнопка нажатие:    scale 1.0 → 0.96, 120ms ease-out
Карточка tap:      elevation 1 → 3, 150ms
Чекбокс:           scale 0 → 1.2 → 1.0, 250ms spring
FAB появление:     scale 0 → 1, 300ms spring + rotate 0 → 0
Загрузка:          Shimmer-эффект (#F0F0F0 → #E0E0E0)
QR-сканер луч:     сверху вниз, 2s linear, loop
Success-анимация:  Lottie checkmark (зелёный, 600ms)
Error-анимация:    shake X ±8px, 3 раза, 400ms
```

### 3.3 Pull-to-Refresh

- Индикатор: логотип SoftTime (play-иконка, синяя, rotating)
- Цвет: #1877F2

### 3.4 Skeleton Loading

- Цвет базы: #F3F4F6
- Shimmer-цвет: #FFFFFF с opacity 0.7
- Время цикла: 1.5s
